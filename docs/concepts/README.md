# Per-Exercise Concept Guides

This directory contains focused concept guides for each exercise. Each guide includes:

- **Key Concepts** (3-5 main takeaways)
- **Visual Diagram** (Mermaid or ASCII)
- **Code Examples** (minimal, focused)
- **Teach-Back Prompt** (test your understanding)

## How to Use

1. Read the concept guide **before** starting the exercise
2. Keep it open as a reference while coding
3. Use the "Teach-Back" section after completing the exercise
